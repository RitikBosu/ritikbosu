## 🚀 Projects

### [Scrolla — Mindful Social Media Platform](https://github.com/RitikBosu/Scrolla)
> MongoDB · Express.js · React.js · Node.js · Cloudinary · WebSockets

- Reduced average user session length by **40%** through request-lifecycle session-limit enforcement
- Cut database round-trips by **90%** by replacing unbounded arrays with normalised collections and `$in` batch queries — eliminating N+1 query chains
- Built a concurrent media pipeline processing up to **5 files (≤100 MB each)** via Cloudinary signed uploads and a custom React editor
- Designed a publicId-based storage layer for instant URL-driven transformations with zero re-encoding
